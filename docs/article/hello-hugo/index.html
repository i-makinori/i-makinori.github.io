HugoとGitHub Pagesによる継続的な文書の編集と公開。

備忘録

コンテンツの作成

$ hugo new <path/of/file_name>

例、articleディレクトリの下にhello-hugo.mdを作成する。

$ hugo new /article/hello-hugo.md

コンテンツの更新

原稿の更新が適用され、公開用の静的ファイルが生成される。

$ hugo
